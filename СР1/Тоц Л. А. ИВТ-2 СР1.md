## Задание 1.4

### Развертывание MongoDB с помощью Docker

## 1. Установка и запуск MongoDB

Команда запуска контейнера MongoDB:

```bash
docker run -d \
--name mongodb \
-p 27017:27017 \
mongo
```

### Описание команды:

- `docker run` — запуск контейнера;
    
- `-d` — запуск в фоновом режиме;
    
- `--name mongodb` — имя контейнера;
    
- `-p 27017:27017` — проброс порта MongoDB;
    
- `mongo` — используемый образ MongoDB.
    

---

## 2. Проверка запущенного контейнера

Команда:

```bash
docker ps
```

Пример результата:

```bash
CONTAINER ID   IMAGE   STATUS         PORTS
a1b2c3d4e5     mongo   Up 2 minutes   0.0.0.0:27017->27017/tcp
```

---

## 3. Подключение к MongoDB

Команда:

```bash
docker exec -it mongodb mongosh
```

---

## 4. Создание базы данных и коллекции

```javascript
use university
db.students.insertOne({

```

## 5. Получение данных из коллекции
